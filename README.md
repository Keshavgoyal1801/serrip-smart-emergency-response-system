# SERRIP - Smart Emergency Response & Intelligent Routing Platform

A full-stack emergency dispatch and intelligent routing platform built using **Java, Spring Boot, Spring Security, JWT, MySQL, React, Graph Algorithms, and Priority Queues**.

SERRIP is designed to simulate a real-world emergency response system that efficiently manages emergency requests, hospitals, ambulances, dispatch operations, and intelligent route optimization while maintaining secure authentication and authorization.
