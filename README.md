# SERRIP - Smart Emergency Response & Intelligent Routing Platform

A full-stack emergency dispatch and route optimization system built using Java, Spring Boot, MySQL, React, Graph Algorithms, and Priority Queues.

## Current Features

- Emergency Request Management
- REST APIs
- MySQL Integration
- Layered Spring Boot Architecture
- GitHub Version Control

## Why This Project?

Most student projects only perform CRUD operations.

SERRIP goes beyond CRUD by implementing:

- Real-time emergency request management
- Ambulance allocation logic
- Hospital selection algorithms
- Graph-based city road network
- Dijkstra shortest path routing
- Priority Queue based dispatch optimization
- Dynamic road blockage and traffic updates
- Full-stack architecture using Spring Boot + React
- Dockerized deployment

This project demonstrates:
- Data Structures & Algorithms
- System Design
- Backend Development
- Database Design
- REST API Development
- Frontend Development
- DevOps Fundamentals
